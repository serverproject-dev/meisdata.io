Meisdata - Logic matters

Linked Data und Dezentralisierung praktisch.
Solid ist eine globale Datenbank, die die Daten von den Anwendungen trennt.

Als Pod-Provider stelle ich Ihnen eine WebId und einen Personal Datastore zur Verfügung.
Sie als User werden in den Mittelpunkt gestellt und alle Hoheitsrechte über Ihre Daten bleiben bei Ihnen.

was kann ich für Sie tun ?

- 50 MB Quota
- Ihr Ticket ins Web 3.0
- eine weltweit einmalige ID
- das Ecosystem wächst stetig
- eine lebendige Community
- Interoperabilität und eine große Auswahl an Anwendungen
- ein offener Standard und Spec-Compliance
- das Recht zum Erstellen, Bewegen, Löschen
- das letzte Login, das Sie brauchen werden
- Support und Begleitung
